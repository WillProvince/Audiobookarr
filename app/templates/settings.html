Library Organisation

When a download completes, Audiobookarr will move the audio files into your audiobook library using the naming format below.

Destination folder for organised audiobooks (as seen by the Audiobookarr container). Leave blank to disable automatic file organisation.
Supported tokens: {author}, {title}, {series}, {series_index}, {narrator}, {year}.
Examples:
• {author}/{title} → Terry Pratchett/Guards Guards
• {author}/{series}/{series_index} - {title} → Terry Pratchett/Discworld/8 - Guards Guards
• {author}/{title} ({year}) → Terry Pratchett/Guards Guards (1989)
Note: tokens for fields that are empty or not set will be replaced with an empty string. If the resulting path has consecutive separators (e.g. //), they will be collapsed.
